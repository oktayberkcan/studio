# Royal Cuts | Premium Men's Grooming

Bu proje, modern bir berber dükkanı için hazırlanmış, Next.js 15, Tailwind CSS, ShadCN UI ve Firebase entegrasyonuna sahip profesyonel bir web sitesidir.

## Özellikler

- **Modern & Premium Tasarım:** Space Grotesk ve Inter fontları ile şık bir görünüm.
- **Online Randevu Sistemi:** Firebase Firestore entegreli randevu formu.
- **Dinamik Fiyat Listesi:** Şeffaf ve yönetilebilir hizmet tarifesi.
- **Stil Portfolyosu:** Yapılan işlemlerin sergilendiği modern galeri.
- **Mobil Uyumlu:** Tüm cihazlarda kusursuz çalışan duyarlı tasarım.

## Başlangıç

Projeyi yerel bilgisayarınızda çalıştırmak için aşağıdaki adımları izleyin:

1. **Bağımlılıkları Yükleyin:**
   ```bash
   npm install
   ```

2. **Çevre Değişkenlerini Ayarlayın:**
   `.env.local` dosyası oluşturun ve Firebase yapılandırma bilgilerinizi ekleyin:
   ```env
   NEXT_PUBLIC_FIREBASE_API_KEY=your_api_key
   NEXT_PUBLIC_FIREBASE_AUTH_DOMAIN=your_auth_domain
   NEXT_PUBLIC_FIREBASE_PROJECT_ID=your_project_id
   NEXT_PUBLIC_FIREBASE_STORAGE_BUCKET=your_storage_bucket
   NEXT_PUBLIC_FIREBASE_MESSAGING_SENDER_ID=your_sender_id
   NEXT_PUBLIC_FIREBASE_APP_ID=your_app_id
   ```

3. **Geliştirme Sunucusunu Başlatın:**
   ```bash
   npm run dev
   ```

4. **Tarayıcıda Açın:**
   `http://localhost:9002` adresine gidin.

## Teknolojiler

- **Framework:** Next.js 15 (App Router)
- **Styling:** Tailwind CSS
- **UI Components:** Radix UI & ShadCN
- **Backend:** Firebase (Firestore & Auth)
- **AI Integration:** Google Gemini (Genkit)
- **Icons:** Lucide React
