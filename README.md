# Royal Cuts | Premium Men's Grooming

Bu proje, modern bir berber dükkanı için hazırlanmış, Next.js 15, Tailwind CSS, ShadCN UI ve Firebase entegrasyonuna sahip profesyonel bir web sitesidir.

## Özellikler

- **Modern & Premium Tasarım:** Space Grotesk ve Inter fontları ile şık bir görünüm.
- **Online Randevu Sistemi:** Firebase Firestore entegreli randevu formu.
- **Dinamik Fiyat Listesi:** Şeffaf ve yönetilebilir hizmet tarifesi.
- **Stil Portfolyosu:** Yapılan işlemlerin sergilendiği modern galeri.
- **Mobil Uyumlu:** Tüm cihazlarda kusursuz çalışan duyarlı tasarım.

## Yerel Kurulum Adımları

Projeyi bilgisayarınıza indirdikten (Download ZIP) ve klasöre çıkardıktan sonra şu adımları izleyin:

1. **Terminali Açın:** Proje klasörünün içinde bir terminal (CMD, PowerShell veya Terminal) açın.
2. **Bağımlılıkları Yükleyin:**
   ```bash
   npm install
   ```
3. **Geliştirme Sunucusunu Başlatın:**
   ```bash
   npm run dev
   ```
4. **Tarayıcıda Açın:** `http://localhost:9002` adresine gidin.
