# **App Name**: Royal Cuts

## Core Features:

- Home Page UI: A full-screen hero section featuring bold typography, evocative background imagery, and a prominent 'Book Appointment' call-to-action button.
- Services Display: Dedicated page with a responsive card layout, detailing each grooming service (Haircut, Beard Trim, Combo, Skin Care, Manicure) with an icon, description, and duration.
- Pricing Table: A clean and clear pricing table displaying service names and their respective costs on a dedicated page.
- Image Gallery: A 2x2 grid layout on a dedicated page for showcasing before/after transformations and salon interior photographs, using placeholder images initially.
- Online Appointment Form: A dedicated page with an intuitive form for booking services, including fields for full name, phone, email, service type (dropdown), date, and time.
- Appointment Submission Handling: Functionality to securely POST appointment form data to the specified webhook URL upon submission and display a clear confirmation message to the user.
- Firestore Appointment Storage: Secure storage of submitted appointment details (full name, contact information, service type, date, and time) into a Firebase Firestore database.

## Style Guidelines:

- The chosen color scheme is dark, evoking a premium and masculine atmosphere.
- Background color: A very dark, desaturated slate blue-gray, `#141A1F`, to provide a sophisticated base for content.
- Primary color: A light, muted blue-gray, `#98ADBD`, used for primary text, general interface elements, and navigation to contrast effectively with the dark background.
- Accent color: A vibrant electric blue-green, `#26D9CD`, chosen for interactive elements like 'Book Appointment' CTAs and highlights, adding a modern and dynamic touch while being analogous to the primary hue.
- Headlines and prominent display text will use the sans-serif font 'Space Grotesk' for a bold and contemporary feel.
- Body text and general content will utilize the neutral and legible sans-serif font 'Inter' for clear readability.
- Modern and minimalist line icons will be used for service cards and other UI elements to maintain a clean aesthetic.
- The design will be fully responsive and mobile-first, ensuring optimal viewing and interaction across all devices, from hero sections to grid layouts.
- Subtle, smooth transitions and hover effects will be incorporated for a premium and polished user experience, avoiding excessive or distracting movement.
